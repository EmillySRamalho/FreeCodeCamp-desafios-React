# Desafios React - freeCodeCamp ⚛️

Este repositório foi criado com o objetivo de armazenar os desafios, exercícios e pequenos projetos desenvolvidos durante meus estudos de React através da plataforma freeCodeCamp.

Aqui compartilho minha evolução prática na biblioteca React, explorando conceitos fundamentais do desenvolvimento front-end moderno por meio de aplicações simples, funcionais e focadas no aprendizado.

---

## 📌 Sobre o projeto

Os desafios presentes neste repositório foram desenvolvidos como forma de praticar:

- Criação de componentes
- Props
- Manipulação de eventos
- Estados com `useState`
- Renderização de listas
- Formulários
- Estruturas condicionais
- Organização de componentes
- Estilização de interfaces
- Lógica de programação com JavaScript

Cada pasta representa um desafio ou mini projeto independente, permitindo praticar diferentes conceitos do React de forma separada e progressiva.
